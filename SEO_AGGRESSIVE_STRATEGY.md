## 2. On-Page SEO Perfection

### Title Tag Formula
```
[Primary Keyword] - Free Online [Tool Type] | ConvertRocket
```

**Examples**:
- `HEIC to JPG Converter - Free Online Image Tool | ConvertRocket`
- `Compress PDF Online Free - No Size Limit | ConvertRocket`
- `TikTok Downloader No Watermark - Free HD Video | ConvertRocket`

**Rules**:
- **50-60 characters** max (mobile SERP optimization)
- Include **modifiers**: "Free", "Online", "No Watermark", "2026"
- **Brand at end** for recognition

### Meta Description Formula
```
[Action verb] [format] to [format] instantly. [Unique benefit]. [Social proof]. [CTA]. 100% free, no signup.
```

**Examples**:
- `Convert HEIC to JPG instantly. Preserve quality, remove watermarks. Trusted by 1M+ users. Try now - 100% free, no signup.`
- `Compress PDF files up to 90% smaller. No size limits, no quality loss. Process 10,000+ PDFs daily. Start free.`

**Rules**:
- **140-155 characters** (optimal for mobile)
- **Action-oriented** language
- Include **numbers** and **social proof**
- End with **strong CTA**

### Heading Hierarchy
```html
<h1>[Primary Keyword]</h1>
<h2>How to [Action]</h2>
<h3>Step 1: [Specific Action]</h3>
<h2>Why Use [Tool Name]?</h2>
<h3>Benefit 1</h3>
<h2>Frequently Asked Questions</h2>
<h3>Is [tool] safe?</h3>
```

**Rules**:
- **ONE H1** per page (primary keyword)
- **H2s** for main sections
- **H3s** for subsections
- Natural keyword usage, **never stuffed**

1. **H1**: Primary keyword
2. **Tool interface**: Immediate action
3. **Trust signals**: "1M+ conversions", "100% free", "No signup"

## 6. Performance Optimization

### Core Web Vitals Targets
- **LCP (Largest Contentful Paint)**: < 1.5s
- **FID (First Input Delay)**: < 50ms
- **CLS (Cumulative Layout Shift)**: < 0.05

### Implementation
1. **Minified CSS/JS**: All assets compressed
2. **Lazy loading**: Images below fold
3. **Inline critical CSS**: Above-fold styles in `<head>`
4. **No render-blocking JS**: All scripts `defer` or `async`
5. **Optimized fonts**: Google Fonts with `display=swap`

## 10. Disproportionate Ranking Tactics

### For Small Sites with No Authority

#### 1. **Target Zero-Competition Keywords**
- Use Ahrefs/SEMrush to find keywords with KD < 10
- Focus on problem-based queries: "how to", "can't", "won't work"

#### 2. **Dominate Long-Tail**
- Instead of "PDF converter", target "compress pdf online free no size limit 2026"
- 4-6 word keywords = less competition

#### 3. **Rich Results Eligibility**
- FAQ schema → FAQ rich results
- HowTo schema → Step-by-step rich results
- SoftwareApplication schema → App rich results

#### 4. **Speed as Ranking Factor**
- Sub-2s load time = ranking boost
- Mobile-first = priority indexing

#### 5. **Topic Authority via Clusters**
- Create 20+ related pages (HEIC to JPG, PNG to JPG, etc.)
- Google sees topical authority

#### 6. **User Signals**
- Low bounce rate = quality signal
- High dwell time = engagement signal
- CTR from SERP = relevance signal
