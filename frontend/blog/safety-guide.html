Security is the number one concern for anyone uploading files to the internet. Whether it's a personal photo, a legal document, or a financial spreadsheet, you need to know: Where does my file go? and Who can see it?

In this transparency report, we will explain exactly how modern online converters like ConvertRocket operate, the security measures in place, and what "Privacy by Design" actually means for you.

The 3 Pillars of ConvertRocket Security

1. TLS Encryption

All data transferred between your device and our servers is encrypted using bank-grade TLS 1.3. Even if a hacker intercepted the connection, they would only see gibberish.

2. Ephemeral Storage

We use ephemeral (temporary) storage. Your files are not saved to a database. They exist in RAM or temporary cache only for the duration of the conversion.

3. Automatic Deletion

A "Cron Job" (automated script) runs every hour to wipe the temporary directory. Files are typically deleted instantly after you download them.

4. No Human Access

Our server infrastructure is automated. No employee, developer, or third-party has access to view the files uploaded by users.

Common Myths vs. Reality

Myth: "Online converters steal your data to sell it."

Reality: While this might be true for sketchy, ad-riddled sites that ask you to sign up, reputable converters (like ConvertRocket) monetize via non-intrusive display ads, not data mining. Storing user files is expensive! It actually costs us money to keep your files, so we want to delete them as fast as possible.

Myth: "My file stays on the internet forever."

Reality: Once the download link expires (usually within 1 hour), the file is physically overwritten on the disk. It is unrecoverable.
