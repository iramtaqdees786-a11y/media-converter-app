🛠️ Professional Tools Requirement

Some advanced PDF operations (like Grayscale and PDF/A) require Ghostscript to be installed on your system if you are running this locally. If you encounter errors, please follow these steps:

1. Download Ghostscript

Visit the Official Download Page and choose the version for your OS (Windows, Linux, or macOS).

2. Install & Add to PATH

After installation, ensure the 'bin' folder (e.g., C:\Program Files\gs\gs10.x\bin) is added to your System Environment Variables (PATH).

ConvertRocket will automatically detect Ghostscript once it's correctly installed.
